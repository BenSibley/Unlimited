== Description ==

Fast, responsive, beautiful, and accessible. With Unlimited, your content will reach more people, on more devices, around the world. Unlimited's simple and dark design puts the focus on your content, and supports it with clean, reliable code. Use this minimalist blogging theme to setup a website for a magazine, school, or personal journal. As a free WP theme, you can use Unlimited for as many sites as you want.

**Live Demo**

View a fully-setup live demo of Unlimited here: [Live demo](https://www.competethemes.com/demos/?theme=unlimited)

**Features**

Unlimited comes with the following features:

* Custom logo
* Social media icons
* Search icon/input
* Right & Left sidebar layouts
* Comment display controls
* Custom excerpt length
* Show full posts on Blog
* Full-width page template

**Unlimited Pro**

Unlimited also has an upgrade called [Unlimited Pro](https://www.competethemes.com/unlimited-pro/) that comes with the following features:

* Custom colors
* Additional layouts
* Flexible header image
* Google fonts
* Background images
* Background textures

* Additional menus
* More widget areas
* Display controls
* Custom footer text
